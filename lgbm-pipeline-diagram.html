Forecasting Pipeline Overview

Day-ahead hourly energy consumption forecasting for Aneo

Input Features
📅

Calendar Information

When in time are we predicting for?

Hour of day Weekday / weekend Month Season
🌡️

Weather Forecast

Tomorrow's temperature predictions, available at forecast time

Hourly temperature (°C) Per location
📊

Historical Consumption

Past consumption patterns the model can learn from

7 days ago 14 days ago Same weekday last week
⚠️

5-day data lag: Only consumption data from 5+ days ago is available at prediction time

Model
🌳

LightGBM

Gradient-Boosted Decision Trees

Learns patterns from thousands of historical examples by asking a series of simple questions about the inputs

Hundreds of small decision trees work together
Each tree corrects the mistakes of the previous ones
Fast to train, lightweight to run daily
Output
⚡

24-Hour Forecast

Predicted consumption (MW)

One value for each hour of the next day, generated every morning at 10:00

00:00 12:00 23:00

The model trains on data from all available Norwegian price areas
